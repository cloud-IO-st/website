Crazy possibilities

Socialized Services

P2P is all about sharing, and sharing is a social activity. With the power of social cloud computing some amazing possibilities appear ever more feasible. Take for example some socialized computing services where you contribute resources to participate in a collective service. Socialized services are a class of services based on resource sharing, rather than a specific service in itself.

Content Distribution Network

Content distribution is already possible even using dynamic resource host matching. The verified content hosting service built into the Hub already allows safe content distribution across peers dynamically based on user load distribution. We however want to take this to the next level with IPFS or a similar distributed file system as a peer-to-peer socialized cloud infrastructure service.

Be Your Own Google

AppScale has been integrated with Subutai Social to provide a Google App Engine compatible PaaS layer. Applications written and deployed to GAE can also be run on Subutai Social with the AppScale plugin. That means you can write PHP, Python, Java, and Go applications to run on Subutai Social distributing the application across several peers.

Collective Cyber Defence

Socially security exists in society and among nations. Why can't people collaborating across peer networks benefit as well? We already enable sharing threat information and offer dynamic sharing to provide defensive capabilities against DDoS. Analytics on the Hub, in SS, and in the router assist in detection and counter measures. However we would like to go a step further: we are working on a legal framework and protocol to enable offensive capabilities to defend against attackers.

IDEA: Virtual Office

We're very excited about building applications that can run across peer-to-peer cloud infrastructures for distributed startup teams and SME's with several branch offices. Not only do you tie people and SOHO offices together, but you enable them with common back office and productivity applications they need. Native built in social cloud services and our PaaS layer can be utilized to quickly build this ultimate application on Subutai Social.

IDEA: Virtual Electronics and IoT Laboratory

We love electronics and writing applications for gadget on the edge. The only problem is having instant access to them or the tools for working with them. We've envisioned a virtual laboratory where these devices and various tools could be shared to increase the rate of development of software and hardware. We think such a system could especially benefit those students in universities around the world without the budget to have everything on hand.

If you feel crazy enough to get involved with any of those Crazy Possibilities let us know!
